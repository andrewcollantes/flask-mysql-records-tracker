# Records Tracker

A simple Flask + MySQL CRUD web application.

## Features
- Add records
- Edit records
- Delete records
- Search functionality
- Status tracking

## Tech Stack
- Python
- Flask
- MySQL
- Bootstrap

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Create the database and table.

3. Update database credentials in app.py

4. Run:
python app.py
